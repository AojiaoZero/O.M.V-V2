Welcome to Free Online Manga Viewer Script V2

O.M.V is free PHP script, used as a script fro watching Mangas Online Coded with love and passion, using a cool and indented php code in addition of Bootstrap (Hero Theme from Bootswatch.

What is new in V2

  • Fixing some bugs.
  • Changing the bootsrap theme to (Hero) : http://bootswatch.com/superhero/
  • Adding a fixed footer.
  • Adding an additional page to every chapter (Comments Page).
  • Comments are generated with "Disqus" Solution.

How To:

  1. Open config.inc.php:
  2. $omv_base_url -> url where the script installed Ending with "/" (Slash).
  3. $omv_title -> your desired script title.
  4. $disqus_shortname -> your Disqus short name
